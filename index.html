How It's Made

Static Website

The static website is standard HTML, CSS, and JavaScript, along with the Tailwind CSS library for styling and ViteJS as the dev server.

API

The API is a simple NodeJS and Express app with a few routes that query MaxMind's free GeoLite IP database and return a geolocation based on your IP address.

DevOps

Digital Ocean's App Platform serves both the static site and API. It distributes the static site on a CDN. Additionally, it's able to autoscale the API.
